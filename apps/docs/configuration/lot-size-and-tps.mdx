# Lot size and TPs

Lot and TP settings define your risk profile more than any other configuration.

## Before you start

- Confirm account leverage and minimum lot size
- Confirm strategy style (single vs multi-target behavior)

## Lot size setup

1. Open **Account & Configuration** for the target broker account
2. Go to **Risk**
3. Set your lot sizing mode and values
4. Save and verify with a test signal

## TP setup

1. Open **Stops**
2. Set TP split percentages (for example 50/30/20)
3. Confirm behavior matches your strategy
4. Save changes

## Practical guidance

- Start smaller than your manual size until execution is stable
- Keep TP splits simple at first
- Re-test after changing channel mix or account leverage
