# Connect Telegram

TSCopier only uses Telegram access to read channels and groups you are already a member of.

## Before you start

- Install Telegram on your phone
- Make sure your account can receive login codes
- Keep your 2FA password ready (if enabled)

## Steps

1. Open **Channels** in TSCopier
2. Click **Connect Telegram**
3. Enter your phone number
4. Enter the code sent by Telegram
5. If prompted, enter your Telegram 2FA password
6. Wait for connection confirmation

## Common issues

- **Code not received**: check the Telegram app, not SMS
- **2FA failed**: verify cloud password and keyboard language
- **Session expired**: reconnect Telegram, then refresh Channels

## Security note

TSCopier does not read your personal chats. It only processes signal messages from channels/groups you connect for copying.
