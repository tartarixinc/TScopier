# Add and manage channels

After Telegram is connected, you can add channels that your copier should monitor.

## Steps

1. Open **Channels**
2. Click **Add channel**
3. Select a Telegram channel from the picker
4. Link it to one or more broker accounts
5. Save and confirm the channel shows as active

![Channels page](/images/channels-overview.svg)

## Best practices

- Start with one channel and validate execution
- Use clear naming so each channel is easy to recognize
- Keep inactive/test channels disabled to reduce noise

## Common mistakes

- Adding a channel but not linking it to any account
- Channel is linked but the broker account is inactive
- Telegram session expired after initial setup
