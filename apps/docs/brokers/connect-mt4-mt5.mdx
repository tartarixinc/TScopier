# Connect MT4/MT5

Broker connection lets TSCopier execute parsed signals directly on your trading account.

## Before you start

- MT4/MT5 login ID
- Broker server name
- Account password

## Steps

1. Go to **Account & Configuration**
2. Click **Add account**
3. Choose platform (MT4 or MT5)
4. Enter login, password, and server
5. Save and confirm status is active

## What to verify after connecting

- Account appears in your broker list
- Connection badge shows healthy/active
- It is available as a link target in **Channels**
