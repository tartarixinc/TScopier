# Basic vs Advanced

Choose a plan based on number of accounts, channels, and copy features you need.

## Basic

- 1 demo/live account
- 5 signal backtests per month
- 5 Telegram channels
- 3 TPs
- Single trading mode

## Advanced

- 5 demo/live accounts (expandable)
- Unlimited signal backtests
- Unlimited Telegram channels
- Unlimited TPs/SLs
- Single and range trading modes
- Advanced management tools (layering, close worse entries, reverse signal, auto breakeven/close)

## Which plan should I choose?

- Start with **Basic** if you are validating one account and a small number of channels.
- Use **Advanced** if you run multiple accounts/channels or rely on advanced execution rules.
